## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Открыть главное меню |
| `/menu` | Вернуться в главное меню |
| `/profile` | Профиль, тариф и подключённые кабинеты |
| `/orders` | Последние заказы |
| `/profit` | Прибыль и маржинальность |
| `/stocks` | Остатки и риски out-of-stock |
| `/analytics` | Краткая сводка и аналитика |
| `/alerts` | Контроль, ошибки и уведомления |
| `/accounts` | Подключённые кабинеты WB и Ozon |
| `/sync` | Запустить синхронизацию |
| `/subscription` | Подписка и тарифы |
| `/usermenu` | Открыть главное меню (перенаправление) |
| `/admin` | Админ-панель |
| `/tariffs` | Админ: управление тарифами |
| `/promocodes` | Админ: управление промокодами |
| `/settings` | Настройки бота |
| `/low_margin` | Настроить порог низкой маржи |
| `/help` | Помощь по боту |
| `/cancel` | Отмена текущего действия |

## Пользовательские сценарии

### 1. Новый пользователь

```
/start → регистрация → главное меню
```

### 2. Подключение Wildberries

```
Главное меню → 🏪 Кабинеты МП → Подключить Wildberries
→ ввод названия → ввод API-ключа → проверка → сохранение
```

### 3. Подключение Ozon

```
Главное меню → 🏪 Кабинеты МП → Подключить Ozon
→ ввод названия → ввод Client ID → ввод API key → проверка → сохранение
```

### 4. Проверка API-ключей

```
🏪 Кабинеты МП → Проверить WB ключ / Проверить Ozon ключ
```
или
```
⚙️ Настройки → 🔑 API-ключи → Проверить WB ключ / Проверить Ozon ключ
```

### 5. Профиль

```
⚙️ Настройки → 👤 Профиль → изменить email/телефон/часовой пояс
```

### 6. Уведомления

```
⚙️ Настройки → 🔔 Уведомления → включить/выключить
```
или
```
🔔 Контроль → 🔔 Уведомления
```

### 7. Заказы

```
Главное меню → 🛒 Заказы → любой пункт
```

### 8. Финансы

```
Главное меню → 💰 Финансы → прибыль, план/факт, безубыточность, убыточные заказы
```

### 9. Товары

```
Главное меню → 📦 Товары → себестоимость, Excel-шаблон, МРЦ, остатки
```

### 10. Поддержка

```
Главное меню → 🆘 Поддержка → новое обращение / мои обращения
```

### 11. Администрирование

```
Главное меню → 🛠 Администрирование → все разделы
```
Доступно только пользователям из списка `ADMIN_TELEGRAM_IDS`.

## Техническая архитектура

- **Клавиатуры:** `app/bot/keyboards/main.py`
- **Обработчики:** `app/bot/handlers/*.py`
- **Роутинг:** aiogram 3 Router с приоритетом регистрации
- **FSM-состояния:** `app/bot/states.py`

### Порядок регистрации роутеров

1. `navigation` — /start, /menu, /usermenu
2. `accounts` — подключение кабинетов
3. `costs` — себестоимость и товары
4. `mrc_pricing` — МРЦ и акции WB
5. `subscription` — подписки и тарифы
6. `admin_panel` — админ: тарифы, промокоды
7. `commission_admin` — админ: комиссии
8. `wb_logistics_admin` — админ: логистика WB
9. `user_menu` — меню пользователя
10. `common` — всё остальное (fallback)

### Callback-контракт

Все callback_data из клавиатур должны иметь соответствующий обработчик.
Динамические callback разрешены для префиксов:
- `account:`, `sync:`, `summary:`, `orders:`, `profit:`, `control:`
- `low_margin:`, `timezone:set:`, `mrc:`, `subscription:`
- `admin:`, `admin_deploy:`, `admin_tariff:`, `user:`, `order:`
- `ap:tariff:`, `ap:promo:`, `admin_commission:`
